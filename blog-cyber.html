Complete 4-Module Curriculum Breakdown

Module 1: Networking & Linux Fundamentals

Understand TCP/IP protocols, OSI layer auditing, subnetting, Linux command line, system administration, and shell scripting.

Module 2: Vulnerability Assessment & Ethical Hacking

Perform network scanning with Nmap, vulnerability audits with Nessus, exploitation via Metasploit, and password cracking techniques.

Module 3: Web Application & Cloud Security

Audit OWASP Top 10 vulnerabilities (SQL Injection, XSS, CSRF), configure WAFs, and harden AWS/Azure cloud perimeters.

Module 4: SOC Operations, SIEM & Incident Response

Monitor security event logs using Splunk SIEM, configure IDS/IPS rules, conduct malware analysis, and execute incident containment protocols.
