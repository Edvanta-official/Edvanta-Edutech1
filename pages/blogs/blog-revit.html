Complete 4-Module Curriculum Breakdown

Module 1: Revit Fundamentals & Architectural Elements

Set up BIM project grids, levels, exterior/interior walls, doors, windows, curtain wall systems, floors, and roofs.

Module 2: Structural Modeling & Parametric Families

Model concrete/steel foundations, columns, structural beams, stairs, railings, and build custom 3D parametric component families.

Module 3: Material Takeoffs, Schedules & Annotation

Generate automated door/window schedules, wall material cost takeoffs, room area tags, dimensions, and callout detail drawings.

Module 4: Clash Detection & Navisworks Coordination

Export Revit models into Navisworks, run interdisciplinary clash detection reports between Architectural, Structural, and MEP systems.
